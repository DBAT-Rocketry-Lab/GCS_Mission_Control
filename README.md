# GCS_Mission_Control
Centralized Ground Control Station GUI for Launch tests and Static Fires

Currently this repo has two distinct project directions:
1. Launch GUI -> comprises of real time telemetry data from the rocket in flight, and a live video feed from the rocket's onboard camera. This GUI is designed to provide real-time information and visual feedback during the launch phase.
2. Static Fire GUI -> focuses on monitoring and controlling the static fire test of the rocket's solid boosters. 

We aim to integrate both GUIs into a single, cohesive application that can seamlessly switch between launch and static fire modes, providing comprehensive control and monitoring capabilities for all phases of the rocket's testing and launch operations.
